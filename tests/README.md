# Integration Tests

The cross-crate integration tests for the HiveFabric gateway stack were previously
maintained in the **hivefabric/hive-gateway-tests** repository (now archived, last
pushed 2026-05-26).

That repo contained end-to-end and cross-crate tests covering the full gateway
stack — tenant gateway, comb agent enrollment, capability routing, and LLM provider
dispatch.

Before writing new integration tests here, review the archived repo for patterns and
coverage:

  https://github.com/hivefabric/hive-gateway-tests
